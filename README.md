## 사용법

HTML 파일을 브라우저에서 직접 열면 된다. 외부 의존성 없는 단일 파일이라 오프라인에서도 동작한다.

## 유지보수 규칙

- HTML 수정 의뢰가 있으면 적용하고 git 커밋 + push까지 해서 최신 상태로 유지한다.
- push하면 GitHub Pages가 자동 배포되어 공유 링크가 항상 최신 상태를 가리킨다.
- 반드시 한국어·일본어 2개 언어 대응을 전제조건으로 한다.
