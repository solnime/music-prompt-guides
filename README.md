## 사용법

HTML 파일을 브라우저에서 직접 열면 된다. 외부 의존성 없는 단일 파일이라 오프라인에서도 동작한다.

## 유지보수 규칙

- HTML 수정 의뢰가 있으면 적용하고 git에 커밋까지 해서 최신 상태로 유지한다.
- 커밋 후 같은 아티팩트 URL에 재배포해서 공유 링크가 항상 최신 상태를 가리키게 한다 (URL 매핑은 CLAUDE.md 참고).
- 반드시 한국어·일본어 2개 언어 대응을 전제조건으로 한다.
